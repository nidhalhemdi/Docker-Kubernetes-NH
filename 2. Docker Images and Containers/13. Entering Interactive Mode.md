# ğŸ›ï¸ **4. Interactive Mode: -i and -t**

To let the container **receive input**, use:

### `-i` = keep STDIN open

### `-t` = allocate a terminal (TTY)

Combined:

```

```

Now you can type inside the container app:

```

```

After finishing, the container stops automatically (normal for CLI apps).
