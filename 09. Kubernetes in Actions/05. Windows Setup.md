## â–¶ï¸ **6. Start Minikube (Create the Local Cluster)**

Use the driver matching your hypervisor:

### Example for VirtualBox:

```

```

### Example for Hyper-V:

```

```

Notes:

* If VirtualBox fails due to Hyper-V being enabled, disable Hyper-V **or** just use `--driver=hyperv`.
* Use Administrator PowerShell if required.

Minikube will create:

* A local VM
* A Kubernetes node (master+worker combined)
* All essential Kubernetes components inside the VM

This process may take a few minutes.
